1 Скачать и распаковать архив в удобное для Вас место в Вашей файловой системе.

2 Создать таблицу в базе данных скриптом из файла scriptToDatabase

3 В директории src\main\resources найти файл под названием config.properties в нем:
  1 Установить директорию в которой будут лежать обрабатываемые xml файлы (формат записи должен быть такой: D:\\test task\\test dir\\monitorigdir)
  2 Установить директорию в которой будут лежать уже обработанные xml файлы (формат записи должен быть такой: D:\\test task\\test dir\\handled dir )
  3 Установить директорию в которой будут лежать xml файлы, которые не смогли обработать (формат записи должен быть такой: D:\\test task\\test dir\\failuredir)
  4 Установить период мониторинга (формат записи должен быть такой: 10)
  5 Установить URL к базе данных (формат записи должен быть такой: jdbc:postgresql://127.0.0.1:5432/test)
  6 Установить имя пользователя базой данных(формат записи должен быть такой: postgres)
  7 Установить пароль к базе данных (формат записи должен быть такой: 1111)

4 В папке lib находиться jar файл программы, run.bat - исполняемый файл прогрыммы для Windows, runLinux.sh - исполняемый файл для Linux
5 Собрать проект gradle-ом, в папке libs появиться jar файл, проверить название, он должен называться - XmlHandlerApplication-all-1.0-SNAPSHOT.jar 
6 Запускаем исполняемый файл

Формат входного XML-файла:
<Entry>
<!--строка длиной до 1024 символов-->
<content>Содержимое записи</content>
<!--дата создания записи-->
<creationDate>2014-01-01 00:00:00</creationDate>
</Entry>
